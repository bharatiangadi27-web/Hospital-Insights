🏥 Hospital Insights Dashboard

📌 Project Overview

This project presents an interactive Hospital Analytics Dashboard developed using Excel to analyze hospital performance, patient trends, and revenue insights.
The objective is to transform raw healthcare data into meaningful business insights that support operational and financial decision-making.

📊 Key Metrics
💰 Total Revenue: ₹15.24 Cr
👥 Total Patients: 4,000
🛏️ Average Stay Duration: 6.53 Days
📈 Revenue per Patient: ₹38,118

🔍 Dashboard Features
👨‍⚕️ Doctor-wise Patient Distribution
⚖️ Gender Distribution Analysis
📆 Monthly Revenue Trend
🌆 City-wise Patient Analysis
🏢 Revenue by Department
💳 Payment Mode Breakdown (Card, Cash, Insurance, UPI)
🎛️ Interactive Filters (Doctor & Department)

🛠 Tools Used
Microsoft Excel
Pivot Tables
Charts & Visualizations
Slicers
KPI Cards
Data Modeling

🎯 Business Value
This dashboard helps:
Monitor hospital revenue performance
Identify high-performing departments
Track patient distribution trends
Analyze payment behavior
Support data-driven hospital management decisions

📁 Dataset
The dataset simulates real-world hospital operations including patient records, department details, doctor allocation, payment modes, and revenue data.
